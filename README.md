# Products Management

O Products Management é um sistema web para gerenciamento de produtos e usuários de um supermercado. O projeto é dividido em um Backend e uma aplicação frontend, cada uma documentada separadamente.

Este repositório possui duas partes principais, cada uma com suas próprias instruções de instalação, configuração e execução.

## Onde encontrar as instruções

| Parte do projeto | Documentação |
| --- | --- |
| Backend | `api/README.md` |
| Frontend | `app/README.md` |

## Ordem recomendada

1. Consulte `api/README.md` para configurar e iniciar o backend.
2. Consulte `app/README.md` para configurar e iniciar o frontend.
